1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that allows developers to interact with and manipulate the structure, content, and style of a webpage.

2. How do you select an element from the DOM?

You can select an element from the DOM using JavaScript methods like:

  1. document.getElementById("id") – Selects an element by its ID.
  2. document.querySelector("selector") – Selects the first matching element using CSS selectors.
  3. document.getElementsByClassName("className") – Selects all elements with a specific class.
  4. document.getElementsByTagName("tagName") – Selects all elements with a specific tag name.
3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to manage events from its child elements. Instead of attaching event listeners to each individual child, the parent element listens for events that "bubble up" from the children.

4. How do you manipulate an element's attributes and styles using the DOM?

In the DOM, you can manipulate attributes using setAttribute(), getAttribute(), and removeAttribute(). For example, you can set an image source with element.setAttribute("src", "image.jpg"). To style elements, use the style property, like element.style.color = "red", or remove a style with element.style.removeProperty("color"). These methods enable dynamic updates to attributes and styles of elements.
